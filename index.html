    Introduction

    Here the API blocks are listed to Angular Blog template. All APIs used in the Application are listed here.

    We Used Sails.js to run the API. You can simply follow the online Video tutorial listed below.

    To run the API on Heroku follow this video Run Sails on Heroku.

    To run the API on local follow this video Run Sails on Localhost.

    Feel free contact us at devecha.abid@gmail.com

    API listing

    The below API listing is done for the BlogSterl Website. After hosting the Application you can use their provided URL in front of each below given API

    Please check below for Postman files to directly check the API in postman. The url is configured in Postman xml so as to access localhost only. To check it live you need to replace localhost by URL provided by heroku.

    API server must be running for this API to work properly.

    Reset Password

    This endpoint Reset password of a specific User.

    Resets the Password

    HTTP Request

    POST /user/resetpassword?token=<tokenId> 

    { 
  "password" : "new",
  "confirmPassword": "new"
}

    URL Parameters

    Parameter Description
    tokenId The tokenId of the User to verify

    Query Parameters

    Parameter Description
    password Password of the User
    Password Confirm Password of User

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files...just save them to the `includes` folder and add them to the top of your `index.md`'s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The kitten requested is hidden for administrators only.
    404 Not Found -- The specified kitten could not be found.
    405 Method Not Allowed -- You tried to access a kitten with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The kitten requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
